The Box Model

Beside content each box consists of padding, border and margin. Margins don't define real width of element. "width" attribute by default actually sets width of an conten. "box-sizing" attribute lets change this behavior. "border-box" is preferable to use as box-sizing model. It's not inheritable css property. To apply it to all the elements on the page, the "*" selector is used.
Vertical margins collapse. Horisontal margins are cumulativie.
