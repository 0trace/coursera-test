Realtive and Absolute Element Positioning.

Allow to specify precise offset of element relative to it's original position.

Static Positioning

Another way of saying -- Normanl Document Flow. Default setting for all elements, except "<html>"

Relative Positioning

Element is positioned relative to its position in normal document flow.
Offset properties are: top, bottom, left, rigth.
Element is not taken out of normal document flow.

Relatively Positioned Element

Static Positioning

All offsets (top, bottom, right, left) are relative to the position of the nearest ancestor, which has positioning set on it, other than static. Documents are taken from a normal document flow.
